# Number Guessing Game - Java

## Project Overview

This project is a simple Number Guessing Game developed in Java. The program generates a random number between 1 and 100, and the user attempts to guess the correct number. The game provides hints after each guess and tracks the total number of attempts.

## Objectives

* Practice Java programming fundamentals.
* Implement loops and conditional statements.
* Use Java's Random class for number generation.
* Handle user input using the Scanner class.
* Validate user input and provide appropriate feedback.

## Features

* Random number generation between 1 and 100.
* User-friendly game interface.
* Input validation for non-numeric entries.
* Hint system (Too High / Too Low).
* Attempt counter.
* Displays a success message upon correct guess.

## Technologies Used

* Java
* Scanner Class
* Random Class

## Program Flow

1. Generate a random number between 1 and 100.
2. Ask the user to enter a guess.
3. Compare the guess with the target number.
4. Display feedback:

   * Too High
   * Too Low
   * Correct Guess
5. Repeat until the correct number is guessed.
6. Display the total number of attempts.

## How to Run

Compile:
javac NumberGame.java

Run:
java NumberGame
